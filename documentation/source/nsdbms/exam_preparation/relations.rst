Übersicht möglicher Relationen
==============================

+----------------+-------------------+------------+------------+--------------+
|                | independant/      | dependant/ | dependant/ | independant/ |
|                | shared            | exclusive  | shared     | exclusive    |
+================+===================+============+============+==============+
| ROW            | -                 | 1:1        | -          | -            |
+----------------+-------------------+------------+------------+--------------+
| REF            | 1:1               | -          | -          | -            |
|                | 1:n (from n-side) |            |            |              |
+----------------+-------------------+------------+------------+--------------+
| ARRAY          | 1:n REF!          | 1:n ROW!   | -          | -            |
| (fixed length) | n:m REF!          |            |            |              |
+----------------+-------------------+------------+------------+--------------+
| MULTISET       | 1:n REF!          | 1:n ROW!   | -          | -            |
|                | n:m REF!          |            |            |              |
+----------------+-------------------+------------+------------+--------------+
