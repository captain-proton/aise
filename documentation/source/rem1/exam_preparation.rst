Kardinalität vs Multiplizität
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

In der Kardinalität wird nur bis zur Mitte der Beziehung gelesen. Bsp:

Stud (0,*) - hört = Ein Student hört 0 bis beliebig viele ...

Kontext
^^^^^^^

Unterschied ``Development Context`` <-> ``RE Context``

:Subject: Alle Objekte, deren Daten im System abgebildet werden
:Usage: Alle Dienste, die das System benutzen
:IT-System: Alle IT Systeme, die mit dem System kommunizieren

M0 - M3 Hierarchien
^^^^^^^^^^^^^^^^^^^

:M3: Meta-meta-Modelle (Kästen mit Strichen verbunden)
:M2: Meta-Modelle (Ein Kasten ist eine Entität)
:M1: Objekttypen (Person)
:M0: konkrete Objekte (Peter)
