Bei Zündung des Fahrzeugs wird eine Systemprüfung, in der die angebundenen Sensoren geprüft werden, durchgeführt.
Funktionale Anforderung
Beim Schalten in den ersten Gang und einer Geschwindigkeit von <= 10 km/h aktiviert das System die Sensoren in der Frontstoßstange des Fahrzeugs.
Funktionale Anforderung
Das System ist so modular aufgebaut, dass einzelne defekte Sensoren ausgetauscht werden können, ohne das ganze System austauschen zu müssen.
Qualitätsanforderung - Maintainability
Das System soll 0.8 Sekunden nach der Zündung funktionsfähig sein.
Qualitätsanforderung - Performance efficiency
Die Sensoren sollen bei einer Umgebungstemperatur von -25℃ bis 50℃ in ihrer Funktion nicht beeinträchtigt werden.
Rahmenbedingung - Physical
Das System sollte unter einer agilen Vorgehensweise nach Scrum entwickelt werden
